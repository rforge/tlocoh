T-LoCoH Manual Install

1. Install the dependent and suggested packages by typing the following three lines at the R prompt. This will install only packages that are not already installed.

dep.pkg <- c("pbapply", "sp", "FNN", "rgeos", "rgdal", "maptools", "png", "move", "raster", "XML")

pkgs.not.installed <- dep.pkg[!sapply(dep.pkg, function(p) require(p, character.only=T))]

install.packages(pkgs.not.installed, dependencies=TRUE)

2. Package gpclib is a special case because a binary version does not exist for Windows. If you can't get this package to install, don't panic, T-LoCoH will still work in most cases.

## Mac OS
if (!require(gpclib)) install.packages("gpclib")

## Windows
if (!require(gpclib)) install.packages("gpclib", type="source")

3. Download the latest version of the T-LoCoH package below (scroll down the list until you see tlocoh, right-click and select 'Save As...'). Remember where you save the file, but do not unzip or expand it.

  • Windows (binary)
  • Source (Mac and Linux)

Note: R-Forge has discontinued building Mac binaries for the time being.

4. Install the T-LoCoH package from the R Console

  • Windows: Install the package by going to the Packages menu on the R console and selecting Install packages from local zip files.
  • Mac: Select Package Installer, then select Local Binary Files. Click the Install button and pick the *.tgz file you downloaded.
  • Linux: Select Package Installer, then select Local Source Files. Click the Install button and pick the *.tar.gz file you downloaded.

5. Load T-LoCoH

Once T-LoCoH is installed, you can load it into memory by typing:

require(tlocoh)
