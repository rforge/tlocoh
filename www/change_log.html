T-LoCoH Change Log

Version 1.12

  • Added support for rasterized isopleths to 
    summary.locoh.lhs
    and 
    plot.locoh.lhs
    .
  • Add creation functions to the output of 
    hm.expr
    .
  • Created help page for LoCoH-hullset (class 
    locoh.lhs
    )
  • Deprecated 'ud' argument in 
    lxy.lhs
    . Replaced with 
    iso
    argument.
  • Fixed a bug in 
    lhs.exp.shp
    which prevented exporting isopleths by themselves to shapefiles.
  • Fixed a bug in 
    plot.locoh.lhs
    when specifying which isopleth levels to plot.
  • Fixed a bug in 
    xyt.lxy
    which caused an error when the projection system wasn't specified.
  • Other bug-fixes and updates to documentation.

Version 1.1.10

  • renamed 'fn' argument to 'file' in 
    lxy.save
    and 
    lhs.save
    for consistency with other R functions
  • added function 
    lxy.exp.kml
    to export lxy objects to KML for animation in Google Earth

Version 1.0.9

  • Added 
    move.lxy
    function to import movement data from the move package
  • Added 
    lxy.reproject
    function to reproject coordinates
  • Fixed a bug with 
    summary
    functions
  • Added checking for syntax of projection arguments
  • Added argument to function 
    xyt.lxy
    to suppress warnings if coordinates are in lat long
  • Added function 
    lhs.iso.rast
    to save isopleths to a RasterLayer
  • Renamed 
    trim
    funciton to strTrim to avoid conflict names with other packages (e.g., raster)
  • Renamed 
    str.text.contrast.color
    function to conform with S3 methods syntax
  • Updated tutorials
