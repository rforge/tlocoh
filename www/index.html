Overview


Utilzation distribution of a female Springbok antelope in Etosha National Park, Namibia

T-LoCoH (Time Local Convex Hull) is a method for constructing home ranges and exploring spatio-temporal patterns in movement data. It is built upon the LoCoH method but contains with new analytical functions for data that have time values attached (e.g., movement data).

T-LoCoH can analyze any set of point data (with or without time stamps), but it has been tailored for data collected at regular intervals from a GPS device, such as GPS collars used in wildlife tracking studies. T-LoCoH uses the time information from each location to produce models of space-use that have strong fidelity to temporal partitioning strategies.

T-LoCoH can generate home ranges models that differentiate internal space not only by how often the animal was seen there (the traditional approach), but also how the animal used space in terms of time-use patterns and behavior mode. You can think of this as behavior maps.

T-LoCoH is not a one-click solution. To use T-LoCoH effectively, you should understand your research question, have some basic knowledge about the study system, and understand how the method works. The tutorial below does a pretty good job explaining how to use T-LoCoH intelligently for specific types of questions and applications. If you are looking for a quick and dirty home range construction method, we suggest you try the Kernel method or one of the other methods from the AdehabitatHR package.

Package Features

  • import and visualize location data
  • thin 'bursts' of GPS points to reduce over-sampling bias
  • explore effects of the space-time balance used in hull construction
  • create and evaluate hulls and density isopleths (i.e., utilization distributions)
  • compute additional hull metrics for time use and directionality
  • create isopleths that reflect space use or any other of hull metric including ancillary variables (e.g., temperature, heart rate, etc.)
  • create scatterplots of hull metrics and use them as a map legend (see sample on right)
  • create and export animations of the animal's movement, including multiple individuals tracked over the same time period
  • display GIS files and imagery on plots
  • export results in tabular and GIS formats
  • getting started tutorial (see below)

Installation

The documentation for T-LoCoH is still in progress, so the package is not yet in the R-Forge repository and thus can't be installed from the R prompt. Instructions for manual installation are below:

1. Install the dependent and suggested packages by typing the following three lines at the R prompt. This will install only packages that are not already installed.

dep.pkg <- c("pbapply", "sp", "FNN", "rgeos", "rgdal", "maptools", "png")

pkgs.not.installed <- dep.pkg[!sapply(dep.pkg, function(p) require(p, character.only=T))]

install.packages(pkgs.not.installed, dependencies=TRUE)

2. Package gpclib is a special case because a binary version does not exist for Windows. If you can't get this package to install, don't panic, T-LoCoH will still work in most cases.

## Mac OS
if (!require(gpclib)) install.packages("gpclib")

## Windows
if (!require(gpclib)) install.packages("gpclib", type="source")

3. Download the correct version of the T-LoCoH package below (right-click and select 'Save As...'). Remember where you save the file, but do not unzip or expand it.

  • Windows: tlocoh_1.0.7.zip
  • Mac and Linux (source): tlocoh_1.0.7.tar.gz

4. Install the T-LoCoH package from the R Console

  • Windows: Install the package by going to the Packages menu on the R console and selecting Install packages from local zip files.
  • Mac: Select Package Installer, then select Local Source Files. Click the Install button and pick the *.tar.gz file you downloaded.

5. Load T-LoCoH

Once T-LoCoH is installed, you can load it into memory by typing:

require(tlocoh)
